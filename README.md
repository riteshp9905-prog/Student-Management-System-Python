# 🎓 Student Management System

A professional Student Management System built using Python and SQLite.

## Features

- Admin Login
- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Topper Student
- Total Students
- Marks, Percentage & Grade
- SQLite Database
